# ZAWAJ AI — سياق المشروع الكامل

## التقنيات المستخدمة
- Next.js + TypeScript (TSX)
- Supabase (Auth + Database)
- Tailwind CSS + Inline Styles
- خط Cairo من Google Fonts
- التطوير على VS Code محلياً على localhost:3000

## هيكل المشروع
```
app/
  page.tsx              ← صفحة الاستقبال
  layout.tsx            ← الإطار العام (Cairo + RTL)
  globals.css           ← الستايلات العامة
  onboarding/
    page.tsx            ← نموذج التسجيل 4 مراحل
  home/
    page.tsx            ← الصفحة الرئيسية (مؤقتة)
  admin/
    page.tsx            ← لوحة الإدارة
components/
  Brand.tsx             ← شعار التطبيق
  GoogleButton.tsx      ← زر تسجيل الدخول
supabaseClient.ts       ← إعداد Supabase
.env                    ← مفاتيح Supabase وجوجل
```

## جداول Supabase
- **profiles** — بيانات المستخدمين الكاملة
- **site_settings** — إعدادات التطبيق (اسم، لوغو، ألوان)
- **reports** — البلاغات بين المستخدمين
- **interests_options** — قائمة الاهتمامات والهوايات

## ما تم إنجازه ✅
1. صفحة الاستقبال — تصميم رومانسي قرمزي زجاجي مع تسجيل دخول جوجل
2. Onboarding — 4 مراحل كاملة مع التحقق من البيانات والحفظ في Supabase
3. لوحة الإدارة — إحصائيات + إعدادات + مستخدمون + بلاغات + اهتمامات
4. layout.tsx — خط Cairo + RTL عربي
5. globals.css — ثيم قرمزي + تأثير زجاجي + خلفية رومانسية
6. ربط Supabase مع Google Auth يعمل على localhost

## التصميم العام
- خلفية: تدرج بنفسجي قرمزي داكن ودافئ
- اللون الأساسي: #800020 (قرمزي ملكي)
- تأثير: زجاجي ثلاثي الأبعاد (backdrop-blur)
- اتجاه: RTL عربي كامل
- خط: Cairo

## مخطط التطبيق الكامل

### الصفحة الرئيسية (app/home) — لم تُبنَ بعد
- شريط علوي: اسم التطبيق يسار + ثلاث شرطات يمين تفتح قائمة
- القائمة تحتوي: الملف الشخصي، من زارني، المعجبون، سجل الإعجابات، المحظورون، GOLD، الإعدادات، السياسات، تسجيل خروج
- شريط سفلي: رسائل + إشعارات + بحث + الرئيسية
- المحتوى: بطاقات مقترحة بنظام Tinder مع طبقة ضبابية على الصور

### نظام الرسائل
- 3 رسائل مجانية يومياً
- رسائل غير محدودة عند الإعجاب المتبادل

### نظام GOLD
- مستقبلاً: اشتراك شهري أو سنوي
- حالياً: مجاني عند إكمال البيانات + دعوة 3 أشخاص
- مزايا: كشف الصور، وضع خفي، ضعف الظهور في الاقتراحات، علامة ذهبية
